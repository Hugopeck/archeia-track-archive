## Context

The `/archeia` write path generates Layer 3 docs (ARCHITECTURE.md, STANDARDS.md, GUIDE.md) from repo evidence. Before calling this stable, we need to identify:

- Template quality issues or ambiguities that produce inconsistent output
- Missing validation steps or weak self-validation rubrics
- Edge cases the skill doesn't handle (monorepos, non-standard layouts, empty repos)
- Plugin packaging gaps (plugin.json completeness, README accuracy)
- Any remaining coupling to Layer 2/Layer 1 that should be made optional or removed for a Layer-3-only release
