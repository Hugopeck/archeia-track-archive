# Work Items

All canonical operational work items live in `.track/`. The root markdown views are generated projections:

- [`PROJECTS.md`](PROJECTS.md) — portfolio and project rollups
- [`TASKS.md`](TASKS.md) — global task index
- [`BOARD.md`](BOARD.md) — kanban view

Use `/track:list` or `/track:board` to inspect the current backlog and regenerate the views with `bash scripts/track-build.sh` when needed.

See `.archeia/ROADMAP.md` for strategic direction and `docs/ONTOLOGY.md` for
the governance model behind the repo.
