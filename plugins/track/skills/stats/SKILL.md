# Track Stats

## Overview

Report workflow health metrics from Track task files.

## Workflow

1. Read `.track/PROTOCOL.md` if you haven't already this session.
2. Read `.track/index.json` or task files directly.
3. Report: total tasks, open vs closed, by-status, by-priority, by-project, available count.
4. Apply filters if requested: project, cycle.

## Response

Concise health summary with counts and breakdowns.
