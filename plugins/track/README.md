# Track Plugin

File-based multi-agent coordination for Claude Code.

Commands provided by this plugin:

- `/track:init`
- `/track:new`
- `/track:move`
- `/track:show`
- `/track:list`
- `/track:board`
- `/track:stats`
- `/track:claim`
- `/track:release`
- `/track:available`
- `/track:validate`
- `/track:decompose`
- `/track:plan`
